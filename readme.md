# VeriNews

A community-driven news credibility platform where users can submit news articles and collaboratively evaluate their credibility through ratings, comments, and moderated reviews.

## Features

- Custom user authentication
- News article submission
- Credibility verification workflow
- Moderator review system
- Community credibility voting
- Comments system
- Article reporting system
- Personal and staff dashboards
- Search and filtering
- REST API with authentication
- Interactive API documentation with Swagger and ReDoc

## Tech Stack

- Django 5
- Django REST Framework
- drf-spectacular (OpenAPI/Swagger documentation)
- SQLite (development database)
- Bootstrap 5

## Future Roadmap

- AI-assisted credibility analysis
- AI-generated article summaries
- Claim extraction and verification assistance
- Advanced credibility scoring
